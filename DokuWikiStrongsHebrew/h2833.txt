====== H2833: חֹשֶׁן (chôshen) ======

===== Source =====

from an unused root probably meaning to   or  ;

===== Meaning =====

contain sparkle perhaps a  pocket  (as holding the Urim and Thummim), or  rich  (as containing gems), used only of the  gorget  of the highpriest

===== Definitions =====

  - contain
  - sparkle
  - pocket
  - rich
  - gorget

===== Usage =====

breastplate.
