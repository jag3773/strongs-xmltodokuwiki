====== H5646: עָב (ʻâb) ======

===== Source =====

or   ; from an unused root meaning to cover; properly, equivalent to  H5645 ; but used only as an architectural term,

===== Meaning =====

an  architrave  (as shading the pillars)

===== Definitions =====

  - architrave

===== Usage =====

thick (beam, plant).
