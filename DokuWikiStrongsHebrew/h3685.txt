====== H3685: כְּסִיל (Kᵉçîyl) ======

===== Part of Speech =====

n-m

===== Source =====

the same as [[en:strongs:h3684|H3684]];

===== Meaning =====

any notable **constellation**; specifically **Orion** (as if a **burly** one)

===== Definitions =====

  - constellation
  - Orion
  - burly

===== Usage =====

constellation, Orion.
