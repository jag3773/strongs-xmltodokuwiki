====== H4557: מִסְפָּר (miçpâr) ======

===== Source =====

from  H5608 ;

===== Meaning =====

a  number , definite (arithmetical) or indefinite (large,  innumerable ; small, a  few ); also (abstractly)  narration

===== Definitions =====

  - number
  - innumerable
  - few
  - narration

===== Usage =====

  abundance, account, × all, × few, (in-) finite, (certain) number(-ed), tale, telling,   time.
