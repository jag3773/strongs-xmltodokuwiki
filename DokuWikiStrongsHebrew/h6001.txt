====== H6001: עָמֵל (ʻâmêl) ======

===== Source =====

from  H5998 ;

===== Meaning =====

toiling ; concretely, a  laborer ; figuratively,  sorrowful

===== Definitions =====

  - toiling
  - laborer
  - sorrowful

===== Usage =====

that laboureth, that is a misery, had taken (labour), wicked, workman.
