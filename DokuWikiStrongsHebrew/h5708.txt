====== H5708: עֵד (ʻêd) ======

===== Part of Speech =====

n-f

===== Source =====

from an unused root meaning to **set** a period (compare [[en:strongs:h5710|H5710]], [[en:strongs:h5749|H5749]]);

===== Meaning =====

the **menstrual** flux (as periodical); by implication (in plural) **soiling**

===== Definitions =====

  - set
  - menstrual
  - soiling

===== Usage =====

filthy.
