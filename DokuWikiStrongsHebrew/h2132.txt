====== H2132: זַיִת (zayith) ======

===== Source =====

probably from an unused root (akin to  H2099 );

===== Meaning =====

an  olive  (as yielding illuminating oil), the  tree , the  branch  or the  berry

===== Definitions =====

  - olive
  - tree
  - branch
  - berry

===== Usage =====

olive (tree, -yard), Olivet.
